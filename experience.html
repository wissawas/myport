STRAIGHT LEND

FULL-STACK P2P LENDING SIMULATION PLATFORM

Straight Lend is a web-based simulation platform that
replicates the peer-to-peer (P2P) lending process,
developed entirely for educational purposes (no
real transactions involved).

I was primarily responsible for building the full
functionality of the platform—including frontend
development, backend logic, and database integration.
The UI/UX design and ER Diagram were co-designed
with a teammate.

The system allows users to register as borrowers or
lenders, apply or offer loans, upload verification
documents, and track approval status. An admin
panel supports user validation, interest rate checks,
and transaction monitoring based on Thai financial
regulations.

KEY FEATURES:

  • Role-based login (Borrower / Lender / Admin)
  • Loan offer & application workflows
  • Identity/document upload & admin approval
  • Interest rate capped by law (5% annual)
  • Real-time data storage in MySQL
  • Admin dashboard with approval/risk statistics

TECHNOLOGIES USED:

  • Frontend: HTML, CSS, JavaScript
  • Backend: Node.js, Express.js
  • Database: MySQL (with relational schema)
  • Co-designed: UI & ER Diagram

FAKE NEWS DETECTION WEB APP

CREATED AN AI-DRIVEN WEB APPLICATION TO DETECT AND
PREVENT THE SPREAD OF FAKE NEWS.

The system uses BERT (bert-base-uncased) fine-tuned via transfer learning for binary classification of real vs fake news. I handled the entire development pipeline, including data preparation, model training with NLP techniques, and building a Flask web interface. The model was implemented using PyTorch and Hugging Face Transformers, achieving over 90% accuracy.
The goal is to promote awareness and help users verify news content in real-time before spreading misinformation.
